# Appointy - Doctor Appointment Web App
Developed a full-stack healthcare platform enabling patients to register, authenticate, view doctors, and book or cancel
appointments online using React.js frontend.
Implemented JWT-based authentication and secure session management for patients to protect user data and appointment
actions.
Built RESTful APIs with Node.js and Express.js, and used MongoDB for data modeling and persistence of users, doctors, and
appointment records.
Designed a responsive React.js interface with dynamic profile management, doctor listings, and real-time appointment
updates.
